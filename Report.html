Q1 驗證報導真實性

因為我們是從老師給的資訊查到這個新聞，理所當然我們就從這個新聞作為切入點，但我們直觀覺得此新聞給的結果怪怪的，所以我們深入去探討

新聞算法： 106休退學率

        = (106新增辦理休學人數+106休學而未復學人數+106退學人數) / 106在學學生數
        = (106休學人數+106退學人數) / 106在學學生數
        = (209392+91531) / 1273894
        = 300923 / 1273894
        ~ 0.23622
        ~ 25%

=> 幾乎每4個大專學生就有1人休退學

我們的算法： 106休退學率

        = (106休學人數+106退學人數) / (106休學人數+106退學人數+106在學學生數)
